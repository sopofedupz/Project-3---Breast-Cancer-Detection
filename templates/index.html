SUPPORT VECTOR MACHINE


Support Vector Machine is another supervised learning model that we tested. Although we used SVM for linear classification, it can be used for non-linear classification as well.


You may notice that our SVM model exhibited very similar results compared to our Linear Regression model. This can happen when using SVM for linear classification. As Chandrima D. from Nihilent Analytics Ltd. explains, "If your data is linearly separable...then using SVM with linear kernel will yield pretty much [the] same accuracy as using L2-regularized logistic regression. The reason is: both of them use almost similar loss functions."

DECISION TREE


Among our several other classifiers we chose to make a model using a decision tree. A decision tree is a simple yet powerful classifier with strong predictive potential.


By combining several of these yes/no decisions, a giant tree is formed though a supervised machine learning model where the data is split according to a certain parameter. By splitting our dataset (in which we already know the outcome) we can create two separate volumes of data so that we can see how accurately our model is predicting the data. Additionally, we can see how often the model is making mistakes and even if that mistake was a false positive or a false negative. The results of this analysis are shown below.

K NEAREST NEIGHBOR


Another classifier that was worth considering in our analysis was k nearest neighbor. Simple fine tuning allowed the creation of an over-predicting model.


Even though finding the proper k value can be a challenge, in the search of an over-predicting model it was a useful tool. In our testing, we found that using a k value of 3 allowed for an accuracy of 96.3% while still maintaining a rate of just over 1% false negatives. Even though a k value of 9 produced higher accuracy at 96.8% it increased the rate of false negatives to over 2%, effectively doubling it.

The plots below show this classifier's results using a k of 3.
